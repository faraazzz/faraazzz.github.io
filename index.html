I'm Syed Faraaz Ahmed, a Data & AI Engineer who specializes in designing and shipping production-grade data systems. My work sits at the intersection of cloud data engineering, real-time streaming, and applied AI — from raw ingestion all the way to business dashboards.

I build on AWS as my primary cloud, combining serverless compute (Lambda, Glue), distributed streaming (Kafka, Spark), and AI inference (Amazon Bedrock, Ollama) to create pipelines that are autonomous, observable, and self-correcting.

My ML background covers the full spectrum — regression, computer vision (YOLOv8), and generative AI — so I can bridge the gap between the data platform and the models that run on top of it.

Featured Work

End-to-end systems spanning real-time pipelines, AI agents, ML models, and analytics platforms.

Agentic AI ETL Debugging Pipeline

Serverless ETL pipeline on AWS that auto-detects failures and reasons about root causes using Amazon Bedrock. CloudWatch log events trigger a Debug Agent Lambda that classifies errors via rule-based logic, routes complex failures to Bedrock for AI-assisted root cause analysis, and writes structured RCA reports back to S3 with caching to prevent redundant LLM calls.

AWS Lambda Amazon Bedrock CloudWatch S3 RDS PostgreSQL Agentic AI Python

Serverless Medallion Pipeline — Crypto Analytics

End-to-end streaming analytics platform for live cryptocurrency data. A Confluent Kafka producer polls CoinGecko every 2 seconds for BTC, ETH, and SOL. AWS Glue Streaming ingests and lands raw JSON in S3 (Bronze). A Glue batch job refines records into Parquet/Snappy (Silver). Athena aggregates Gold-layer datasets loaded into Redshift, visualized via QuickSight dashboards.

Apache Kafka AWS Glue Amazon Athena Redshift QuickSight Medallion Architecture PySpark

Real-Time Smart Bank Data Streaming

Production-grade Big Data pipeline for real-time banking analytics. Kafka producers stream transaction events into Spark Structured Streaming, with Elasticsearch as the sink and Kibana for live observability dashboards.

Apache Kafka Apache Spark Elasticsearch Kibana Python

AI Gmail Summarization Pipeline

Automated workflow in Kestra that ingests Gmail messages, processes them through Ollama (local LLM), stores structured summaries in BigQuery via dlt, and pushes digest notifications to Slack.

Kestra Ollama (LLM) BigQuery dlt Slack API Python

Chip Defect Detection — Computer Vision

YOLOv8 + CNN web application for detecting and classifying defects in PCBs and semiconductor wafers. Real-time inference with bounding box visualization for manufacturing quality control.

YOLOv8 CNN PyTorch Python Computer Vision

Real Estate Value Estimation — Zillow

End-to-end ML project predicting Zestimate valuation errors using 9 regression models and extensive feature engineering. Insights visualized via Tableau dashboards for real estate market analysis.

scikit-learn Feature Engineering Tableau Regression Python

NYC Taxi — Data Mining & Prediction

Data mining and predictive modeling of NYC Yellow Taxi trips (2024). Full EDA, feature engineering, and ML model pipeline for trip duration and fare prediction on large-scale trip records.

Data Mining Predictive Modeling Pandas scikit-learn Python

Unpaired Image-to-Image Translation

Unsupervised image translation using the UNIT (Unsupervised Image-to-Image Translation) framework on Monet2Photo and Yosemite summer/winter datasets. GAN-based domain adaptation without paired training data.

UNIT Framework GANs PyTorch Computer Vision Python
